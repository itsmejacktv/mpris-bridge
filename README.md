# MPRIS Bridge

MPRIS Bridge is my attempt at making a utility that spits out the same REST API as [SMTC Bridge by **nutty**](https://github.com/nuttylmao/smtc-bridge) allowing Linux users to use his [ Universal Now Playing ](https://nutty.gg/en-aud/products/universal-now-playing) tool.

> [!IMPORTANT]
> I have never worked with the D-Bus before, I give this like 30% success rate initially.
